# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability within this package, please send an email to Daryl Legion at [daryl1legion@proton.me]. All security vulnerabilities will be promptly addressed.

**Please do not report security vulnerabilities through public GitHub issues, discussions, or pull requests.**

## What to include

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Response Timeline

- **Acknowledgment**: Within 48 hours
- **Assessment**: Within 1 week
- **Fix**: As soon as possible, depending on severity

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.x     | Yes       |
